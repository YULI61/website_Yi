Does More Greenspace Lead to Less Obesity?

Spatial Regression, Urban Health Inequality, GIS Buffer Analysis

The "Greenspace & Childhood Obesity" project focused on London and used QGIS to create buffer zones around green spaces, combined with Python-based OLS and Spatial Error Models to analyze the spatial relationship between green space access and childhood obesity rates. A correlation matrix and VIF analysis were used to identify key factors such as tennis court density and deprivation index. The project integrated multiple urban datasets and used spatial regression and visual mapping to highlight how certain types of green spaces relate to health inequality.

How Does Urban Environment Shape Crime Rates?

Urban Crime Analysis, Machine Learning Modeling, Spatial Visualization

The "Urban Crime Patterns" project focused on crime data across London boroughs and applied Python-based machine learning models, including Random Forest and XGBoost, to analyze how urban density, demographic structure, and socioeconomic factors influence the spatial distribution of crime. The project used area-level aggregation, spatial visualization, and feature importance analysis to reveal how variables like population density, deprivation level, and youth ratio contribute differently to various crime types. Grid-based mapping and classification heatmaps were used to visualize high-risk areas and support spatial decision-making for urban governance and public safety resource allocation.

Airbnb & Rental Prices

Platform Economy & Rent Analysis, Multiple Regression Modeling, Data Visualization & Spatial Interpretation

The "Airbnb & Rental Prices" project explored the impact of short-term rental activity on long-term rental affordability in London by combining Inside Airbnb data with public rental records. Python-based multiple linear regression models were applied, with Box-Cox transformation used to normalize variables and improve model fit. Key predictors such as host density, level of commercialization, and listing type were analyzed in relation to rental prices. Spatial visualization showed a strong overlap between Airbnb hotspots and high-rent areas, suggesting a potential crowding-out effect in the housing market. This was a group project; responsibility included Questions 6 and 7, as well as contributing to the overall research structure and discussion design.

Urban Simulation

Urban Transport Network Analysis, Centrality & Node Vulnerability, Spatial Interaction Modeling & Retail Location

The "Urban Simulation" project used the London Underground as a case study to assess network resilience under node failure scenarios using graph-based network analysis. Topological and weighted networks were analyzed with centrality measures (degree, closeness, betweenness) to identify critical transfer stations and simulate fragmentation by sequential node removal. Betweenness centrality proved most effective for revealing structural weaknesses. In the second part, a weighted flow network based on NUMBAT OD data was used to compare two candidate supermarket locations via an unconstrained gravity model. The model incorporated population size, store area, and travel distance. Under a high-friction scenario (increased distance decay), Option 2 retained more local flows, suggesting greater accessibility resilience. This individual project combined spatial network science and interaction modeling, with strong implications for transport planning, network robustness evaluation, and data-informed retail site selection.

Good Trip in London

Urban Tourism Platform, Interactive Web Mapping, Route Optimization

The "Good Trip in London" project focuses on improving tourism experiences in central London by integrating data from VisitBritain, ALVA, and OpenStreetMap to build a categorized, semantically tagged POI database. It uses the Mapbox Optimization API to generate personalized walking routes based on user preferences. The platform includes multiple interactive modules such as animated particle maps, toggleable line charts, attraction heatmaps, and filtered route planners, supporting different travel modes and time ranges. The frontend is built with React and Mapbox GL JS, enabling real-time interaction and responsive design across devices. This was a group project; main responsibilities included website structure and interaction design, covering all modules except the UK map, statistical charts, and the custom route map. (PC only)

London Attractions Map

Interactive Mapping, Spatial Clustering, Urban Data Visualization

The "London Attractions Map" is a web-based interactive map built with Mapbox GL JS, designed to explore the spatial distribution of tourist attractions across London. It integrates POI data from OpenStreetMap and enables real-time spatial analysis using the DBSCAN clustering algorithm. Users can adjust the epsilon (neighborhood radius) parameter to dynamically modify cluster sensitivity and uncover varying density patterns. The interface offers multiple layers of interaction, including toggling between clustered and raw points, filtering by attraction category, zoom-based cluster refinement, and hover popups with contextual details. The design prioritizes clarity and responsiveness, making it an effective tool for uncovering spatial trends in urban tourism.
